Выгрузка справки в формат JSON

Oct 20, 2025

Обработка выгрузки справки

В приложении, справка хранится в справочнике “Справка”, в макете “Справка” (формат json).

Обработка позволяет выгрузить справку после ее подготовки (выгрузка в md-формате, загрузка в html-формате) в формате json.

файл представляет собой массив структур вида:

  • Код - уникальный код справки;
  • Наименование - наименование формы, откуда была вызвана справка;
  • Заголовок - заголовок справки;
  • Текст - основной текст справки;
  • Дополнительно - дополнительный текст справки;
  • HTML - текст справки в формате HTML.

Пример элемента массива: { “Код”: 6, “Наименование”: “Документ.Израсходовано.Форма.ФормаДокумента”, “Заголовок”: “Расход вещей”, “Текст”: “Тут сохраняются данные по расходу.”, “Дополнительно”: “”, “HTML”: “Тут сохраняются данные по расходу.\n” },
