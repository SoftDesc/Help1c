Общее описание

Чтобы создать справку для приложения, следуйте этим пунктам:

  1. Объединение конфигураций. Выполняется через меню конфигуратора “Сравнить, объединить с конфигурацией из файла…”.
  2. Добавление команды “Справка”.
  3. Преобразование справки в формат json. Выполняется преобразование справки из формата md в формат html, затем справка в формате Html загружается в программу, затем справка выгружается в формате json.
  4. Загрузка готовой справки в программу.
  5. Программное восстановление справки.
