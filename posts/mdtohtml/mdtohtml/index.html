Md to HTML

Oct 20, 2025

MdToHTML.exe

MdToHTML.exe - утилита коммандной строки для преобразования файлов формата md в файлы формата html.

Перед использованием утилиты необходимо выгрузить справку в формате md.

Параметры командной строки: MdToHtml.exe Имя <md-файла>

Структура md-файла - массив структур в формате json:

  • Код - уникальный код справки;
  • Наименование - наименование формы, откуда была вызвана справка;
  • Заголовок - заголовок справки;
  • Текст - основной текст справки;
  • Дополнительно - дополнительный текст справки.

Пример md-фала:

{ “Код”: 6, “Наименование”: “Документ.Израсходовано.Форма.ФормаДокумента”, “Заголовок”: “Расход вещей”, “Текст”: “Тут сохраняются данные по расходу.”, “Дополнительно”: "" },

Выходные каталоги:

  • html - в каталог сохраняются файлы в формате html. Имя файла - код справки из md-файла.
  • posts - в каталоге сохраняется выгрузка отдельных md-файлов для использованием с генератором статических сайтов (hugo).

Если каталогов html и posts не существует, они создаются перед сохранением файлов.
